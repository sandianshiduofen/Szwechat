货物运输协议
请您认真阅读《货物运输协议》，点击【同意】即视为您接受并签署本协议。

甲方： 实际承运服务的单位或个人

乙方： 托运货物的单位或个人

丙方： 北京三真车联信息技术有限公司


依中华人民共合国《合同法》及《邮政法》相关规定，甲\乙\丙三方就<< 货物运输协议>>事宜达成如下约定：

一、 甲方权利和义务：

1. 甲方判定托运货物外包装简陋存在安全隐患的，有权要求乙方加强托运货物包装；

2. 甲方无法送达的货物有权拒绝收货；

3. 甲方对运费收费标准有最终解释权；

4. 甲方需按约定时间内将货物安全送达乙方指定收货地点并交付给收货人；

5. 甲方需对货物的安全和货物完好负责，因甲方原因造成货物损坏、丢失的，应按照协议约定承担赔偿责任；

二、 乙方权利和义务：

1. 乙方自主选择甲方提供的服务；

2. 因甲方责任引发货物损坏、丢失，乙方按协议约定主张货物赔偿的权利；

3. 乙方必须如实、准确填写托运单，因乙方填写错误造成退货或配送不到的，由乙方承担相应责任；

4. 按国家法律规定，乙方所发货物不得夹带国家违禁货物，如因夹带造成损失的由乙方全额承担赔偿责任。造成其他事故的，由乙方承担相应的法律责任；

5. 乙方所发货物需提供完好包装，易碎货物需进行加固包装，因包装不当造成的货物损坏由乙方自行承担；

三、 丙方权利和义务：

1. 丙方在乙方与甲方货物托运业务中提供代收代付服务和财务结算服务；

2. 丙方承担乙方在丙方系统内注册账号资金安全、财务结算及时准确的义务；

3. 丙方承担乙方在丙方系统内注册账号账实相符并提供对账凭证的义务；

4. 丙方承担乙方在丙方系统内个人或单位注册及发货信息保密的义务；

5. 如因丙方原因造成乙方货款损失的，由丙方赔偿乙方。

四、代收货款结算：

代收货款结算：收货方签收托运货物并支付货款成功后，丙方按乙方与甲方约定的服务费率扣除服务费，余下货款实时打入乙方在丙方平台系统内注册的资金账户系统保护期账户中，保护期账户按银行结算时效自动转入乙方可用余额账户，乙方可实时提现。

五、 赔偿责任：

甲方运输过程中造成货物损坏、丢失，应承担赔偿责任。赔偿金额最高为损坏、丢失货物运费金额的3—5倍。

因收货方原因造成的货品未正常签收，甲方有告知乙方的义务。告知乙方后，甲方仅提供揽货后15天内免费存放服务，15天后甲方可收取保管费或在告知乙方后对货物进行自行处理。

因发生自然灾害等不可抗力造成货物无法按期运达目的地时，甲方应及时通知乙方并协商解决，因延迟送达乙方产生损失，甲方不承担责任。如甲方未主动告知乙方，乙方损失均由甲方承担。

六、 纠纷处理方式：

乙方和甲方发生货物运输纠纷的，以协商的方式解决，协商未果时，任何一方有权向甲方所在地人民法院提起诉讼。

乙方和丙方发生账款纠纷的，以协商的方式解决，协商未果时，任何一方有权向丙方所在地人民法院提起诉讼。
